# Roadmap

## Near Term

- Build a functional Ratatui dashboard.
- Implement local encrypted SQLite migrations.
- Add CRUD services for patients and encounters.
- Add structured note templates.
- Add audit log persistence.
- Add BAA-gated AI provider interface.

## Mid Term

- Add better note editor ergonomics.
- Add chart search.
- Add vitals and lab trend charts.
- Add billing-review queues.
- Add signed-note versioning and amendments.
- Add backup and restore.
- Add import/export for deidentified demo records.

## Later

- FHIR import/export adapters.
- Claim file generation experiments.
- Local-only model provider integration.
- Role-based access controls.
- Hash-chained audit log.
- Pluggable storage backends.

## Out of Scope for MVP

- Cloud medical storage.
- Production EHR replacement.
- Automated claim submission.
- Automated diagnosis or billing decisions without human review.
- Certified medical device behavior.
